## Description
This project is test task with logic related to subscribing and publishing to mqtt broker topics.

## Notice
For local purpose its better to start it with docker-compose.yml, for mqtt to be available and configured

## Installation
```bash
$ npm install
```

## Running the app

create .env from .env.example

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
